# Issue: `issue-crafter` サブエージェントの登録漏れ修正

## 概要
`v1.11.0` で追加された `agents/issue-crafter.md` が `gemini-extension.json` に登録されておらず、CLI から認識されない問題を修正する。

## 問題点
- `v1.11.0` の実装において、`gemini-extension.json` の `agents` 配列への追記が漏れていた。
- バージョンが `1.10.0` のままであり、リリース内容と整合性が取れていない。

## 解決策
- `gemini-extension.json` に `agents/issue-crafter.md` を追加。
- `package.json` および `gemini-extension.json` のバージョンを `1.11.1` に更新。
- `npm run build` および `node tools/sync-version.cjs` を実行。

## テスト
- `gemini-extension.json` に `issue-crafter.md` が含まれていることを確認。
- `node tools/sync-version.cjs` が正常に終了することを確認。
